# Popover

The Popover component displays floating content in relation to a target.

The floating content can be closes by clicking outside or pushing the Escape key.

## Basic

Checking the 3 positions:

- `"bottom"` (default)
- `"top"`
- `"cursor"`

## Showing the state

Accessing the `isOpen` state from the toggle button.

The content of the toggle button will change, depending on the state.

<Playground>
  <Root>
    <div style={{height: 300}}>
      <Popover
        content={({close}) => (
          <div
            style={{
              width: 240,
              height: 240,
              display: 'flex',
              alignItems: 'center',
              justifyContent: 'center',
              flexDirection: 'column'
            }}
          >
            <div>
              Popover Content <Button onClick={close}>Close</Button>
            </div>
          </div>
        )}
      >
        {({isOpen, open}) => (
          <Button onClick={open}>Trigger Popover (state: {isOpen ? 'open' : 'close'})</Button>
        )}
      </Popover>
    </div>
  </Root>
</Playground>
