## Root Menu

The same menu as before using `<MenuRoot>`, `<MenuItem>` and `<MenuDivider>` low-level components.

<Playground>
  <Root>
    <MenuRoot>
      <MenuItem onClick={console.log}>Edit</MenuItem>
      <MenuItem onClick={console.log}>Clone</MenuItem>
      <MenuItem disabled>A disabled feature</MenuItem>
      <MenuDivider />
      <MenuItem onClick={console.log}>Delete</MenuItem>
    </MenuRoot>
  </Root>
</Playground>

## Inside a `<Popover>` component.

You have to call `close()` from onClick event handler by yourself, if you want to close the menu item when the user clicks on it.

<Playground>
  <Root>
    <Popover
      content={({close}) => (
        <Menu
          items={[
            {
              label: 'Edit',
              onClick: () => console.log('Edit')
            },
            {
              label: 'Clone',
              onClick: () => console.log('Clone')
            },
            {
              type: 'divider'
            },
            {
              label: 'Delete',
              onClick: () => console.log('Delete')
            }
          ]}
        />
      )}
    >
      {({open}) => <Button onClick={open}>Show menu</Button>}
    </Popover>
  </Root>
</Playground>
