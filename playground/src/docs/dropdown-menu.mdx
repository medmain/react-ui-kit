# Dropdown menu

The `<Dropdown>` component puts together `<Popover>` and `<Menu>` to provide a convenient way to create floating menus displayed when the user pushes a button.

How to close the menu:

- Click again on the button
- Click outside the menu
- Click on a menu item
- Push the Escape key

The icon dislayed inside the button depends on the position props.

If you need more flexibility (to control when the menu is closed for example), use `<Popover>` and `<Menu>` buttons directly.

## Basic

Using the default options: `position="top"` and `alignment="left"`

<Playground>
