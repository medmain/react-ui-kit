# Dropdown menu

## Basic

Using the default options: `position="top"` and `alignment="left"`

<Playground>
  <Root>
    <Row>
      <Col>
        <span />
      </Col>
      <Col>
        <DropdownMenu label={'Actions'} position="top">
          <DropdownItem onClick={() => console.log('Replace')}>Replace</DropdownItem>
          <DropdownItem onClick={() => console.log('Delete')}>Delete</DropdownItem>
        </DropdownMenu>
      </Col>
      <Col>
        <DropdownMenu label={'Disabled menu'} position="top" alignment="right" disabled>
          <DropdownItem onClick={() => console.log('Replace')}>Replace</DropdownItem>
          <DropdownItem onClick={() => console.log('Delete')}>Delete</DropdownItem>
        </DropdownMenu>
      </Col>
    </Row>
  </Root>
</Playground>

## Custom label

Using JSX code for the `label` and hiding the default chevron icons

<Playground>
  <Root>
    <Row>
      <Col>
        <span />
      </Col>
      <Col>
        <DropdownMenu
          label={() => (
            <div style={{display: 'inline-flex', alignItems: 'center'}}>
              <AccountIcon size={24} color={'#888'} />
              <span style={{marginLeft: '.5rem'}}>Kawhi Leonard</span>
            </div>
          )}
          showChevronIcon={false}
          position="bottom"
          alignment="right"
        >
          <DropdownItem onClick={console.log}>My account</DropdownItem>
          <DropdownItem onClick={console.log}>Admin</DropdownItem>
          <DropdownItem onClick={console.log}>Sign out</DropdownItem>
        </DropdownMenu>
      </Col>
      <Col>
        <span />
      </Col>
    </Row>
  </Root>
</Playground>
